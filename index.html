Welcome to Latitude! This is a website designed to analyze weather data based on latitude.

Latitude Visualizations

This website is designed to compare weather data based on latitude. The four (4) types of weather data analyzed and compared on this website are Max Temperature (Fahrenheit), Humidity (%), Cloudiness (%), Wind Speed (mph).

The data was collected by creating a random list of cities using citypy based on randomized latitudes and longitudes. This city list was then used with the Open Weather Map API to collect city weather data on 2021-04-19. The data was then processed with Matplotlib and Pandas. The raw data and visualizations can be found throughout this site.
